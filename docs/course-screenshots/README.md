# 课程配图替代截图

这些截图是根据当前项目页面重新生成的过程截图替代版，用于补齐原先 `output/playwright/` 中已经被清理掉的过程截图。

## 推荐使用

- `pmagic-home-desktop.png`：桌面首屏，适合放在“先搭可替换网页框架”或最终效果。
- `pmagic-hero-desktop.png`：Hero 区域，适合讲视频融合、首屏视觉。
- `pmagic-hero-mobile.png`：移动端首屏，适合讲移动端独立重排。
- `pmagic-metrics-painpoints.png`：指标与痛点区域，适合讲密度、间距和卡片细节。
- `pmagic-density-before-after-comparison.png`：间距和密度的前后对比合成图，适合直接放课件。
- `pmagic-density-before-recreated.png`：复刻的初版松散状态，用于和当前完成版做前后对比。
- `pmagic-density-after-current.png`：当前完成版同一区域，用于和复刻初版对比。
- `pmagic-density-mobile.png`：移动端指标与痛点区域，适合讲移动端密度。
- `pmagic-modules-desktop.png`：五大模块区域，适合讲模块卡片还原。
- `pmagic-hover-motion-card.png`：模块 hover 状态，适合讲动效怎么聊出来。
- `pmagic-case-desktop.png`：案例区域，适合讲真实项目和时间轴。
- `pmagic-whitepaper-desktop.png`：白皮书区域，适合讲后续资源和 CTA。
- `pmagic-footer-cta-desktop.png` / `pmagic-footer-links-desktop.png`：底部 CTA 与 Footer。
